Return Cartesian product of two std::tuples For each value from the first argument combine it with each value from the second argument and return std::tuple with results.

Use sequence Seq1 equal to <0, 1,... , sizeof...(Args1)> to call c_product_line multiple times (with pack expansion). Each call to c_product_line will combine one value from the first argument with each value from the second one. Example:

auto res = c_product_bin(std::make_tuple(1,2), std::make_tuple(3,4));
//res is equal to tuple<tuple<int, int>, tuple<int, int>, tuple<int, int>, tuple<int, int>>
//with values ((1,3), (1,4), (2,3), (2,4))

Combine value given at position I from the first tuple with each value from the second tuple Use sequence Seq2 equal to <0, 1,...

, sizeof...(Args2)> to call c_product_one multiple times (with pack expansion). On each time value at position I from the first tuple is combined with next value from the second tuple. Whole operation creates sizeof...(Args2) tuples which are aggregated into one std::tuple, so the type of returned value is equal to std::tuple of std::tuple's. Example:

auto res = c_product_line<0>(std::make_tuple(1,2), std::make_tuple(3,4,5));
//res is equal to std::tuple<std::tuple<int, int>, std::tuple<int, int>, std::tuple<int, int>>
//with values ((1,3), (1,4), (1,5))

const std::tuple< Args2...> &  y

Combine two values from two tuples together Combine the value at position I from the first argument with the value at position J from the second argument.

If first argument is std::tuple and second argument is std::tuple of std::tuples then it glues value from the first argument to std::tuple at position J from the second argument. Those two scenarios could be represented as follows (make_tuple from std):

c_product_one<0,0>(make_tuple(1,2), make_tuple(3,4)); // results in std::tuple<int,int>(1,3)
c_product_one<0,0>(make_tuple(1,2), make_tuple(make_tuple(31, 41), make_tuple(32,42)); //results
//in std::tuple<int,int,int>(1,31,41)
