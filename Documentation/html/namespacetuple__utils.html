Calculate n-ary Cartesian product over n std::tuple's.

Given n std::tuples T1, T2, T3, ..., Tn it calculates (T1 x (T2 x (T3 x (... x (T(n-1) x Tn))))) where Ti x Tj is calculated by details::c_product_bin.

Example:

auto x = std::make_tuple(6, 3);
auto y = std::make_tuple("xxx", 5.3);
auto z = std::make_tuple(7, -2);
auto res = c_product(x, y, z); //res is std::tuple of std::tuples containing ((6, xxx, 7), (6, xxx, -2),
// (6, 5.3, 7), (6, 5.3, -2), (3, xxx, 7), (3, xxx, -2), (3, 5.3, 7), (3, 5.3, -2))

Folds few tuples into one by calling function f with arguments given on each position from those tuples. Used to produce std::tuple in which each element is equal to the result of recursive invocation of function f for each element from from std::tuples given as an arguments at each index

Template Parameters
FuncType- type of function used for folding
Res...- unknown number of std::tuples types given as arguments
Parameters
f- binary or unary function which will be used to fold one or more tuples
rest- unknown number of std::tuples
Returns
std::tuple in which each position is equal to the result of folding values from the same positions from the tuples given as arguments

Example Usage:

auto tuple_arg1 = std::make_tuple(2, 4.4, "hello ");
auto tuple_arg2 = std::make_tuple(3, 1, "world");
auto result = tuple_utils::fold(std::plus<>(), tuple_arg1, tuple_arg2); //std::plus<>() from C++14
// result is equal to std::tuple<int, double, const char*> result {5, 5.4, "hello world}

Create tuple from existing tuple based on given sequence. Based on the type of tuple given as an argument and sequence specified as a template parameters returns a std::tuple with types and values corresponding to the base tuple. It is possible to use the same sequence more than once.

Template Parameters
Sequence- integers indicating which base tuple values are to be used in constructing custom tuple
Parameters

Merge few std::tuples into one using their values Takes any number of heterogenous std::tuples and returns an aggregate std::tuple that contains each value from the given arguments, in order.

Parameters
tuple1,tuple2,rest...- at least two possibly heterogenous std::tuples
Returns
std::tuple with values from all std::tuples passed as arguments

Example Usage:

auto t1 = std::make_tuple(1.1, 2, '3');
auto t2 = std::make_tuple(4, 5);
auto t3 = std::make_tuple("hello", "world");
auto res = merge(t1, t2, t2); //res is a std::tuple<float, int, char, int, int, const char*, const char*>
//with values {1.1, 2, '3', 4, 5, "hello", "world"};

template<typename... Tuples>
auto tuple_utils::zip ( Tuples &&...  tuples) -> decltype(details::zip_proxy( std::integral_constant<bool, contain_empty<Tuples...>::value>(), std::forward<Tuples>(tuples)... ))

Zip together values at corresponding indexes of arbitrary number of std::tuples For each index create one std::tuple with values taken from std::tuples passed as arguments.

Result will contain tuples created starting from index zero up to index equal to the size of the smalles tuple passed as an argument. Use tag dispatch and details::zip_proxy to check if any std::tuple given as parameter is empty, if so then tuple_utils::zip should return empty std::tuple<> Example:

auto result = tuple_utils::zip(make_tuple(1, 4, 2.2, "test"), make_tuple(2, 2.2, 1), make_tuple(4, 6, 9));
//result is equal to tuple<tuple<int, int, int>, tuple<double, double, int>, tuple<double, int, int>>
//and contain values ((1, 2, 4), ((4, 2.2, 6), (2.2, 1, 9))). Notice that value "test" from the first
//tuple is not used because other tuples have smaller size than the first one.
